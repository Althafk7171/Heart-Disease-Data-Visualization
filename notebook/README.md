# Notebook

This folder contains the Jupyter Notebook used for heart disease data analysis and visualization.

## File

- `Heart_Disease_Analysis.ipynb` - Main notebook containing data loading, exploratory data analysis, and visualizations.

## Notebook Contents

The notebook includes:

- Importing required Python libraries
- Loading the heart disease dataset
- Dataset overview
- Age distribution analysis
- Categorical feature visualization
- Violin plots
- Correlation heatmap
- Pairplot
- Jointplots
- Summary of visual insights

## How to Run

Open the notebook using Jupyter Notebook or JupyterLab and run all cells.
