# Data

This folder contains the dataset used for the Heart Disease Data Visualization project.

## File

- `heart.csv` - Heart disease dataset containing patient health records.

## Dataset Details

The dataset includes health-related attributes such as age, sex, chest pain type, resting blood pressure, cholesterol, fasting blood sugar, resting ECG, maximum heart rate, exercise angina, oldpeak, ST slope, and heart disease status.

## Note

This dataset is used for exploratory data analysis and visualization practice.
